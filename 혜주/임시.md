# 3. 인덱스를 이용한 소트 연산 생략

 

## ORACLE ROWNUM

- P.357
```SQL
SELECT *
  FROM EMP
 WEHRE ROWNUM < 3 --가능
 WHERE ROWNUM > 3 --불가능
 WHERE ROWNUM BETWHEEN 3 AND 5 --불가능
 ```
