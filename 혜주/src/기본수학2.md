# 기본수학2

## 소수찾기
### 1. 자기자신보다 작은 수로 나누기
```java
for(int i=2; i<num; i++) {
	if(num % i ==0) {
		result = false;
		break;
	}
}
```
### 2. 제곱근
```java
for(int i=2; i<=Math.sqrt(num); i++) {
	if(num % i ==0) {
		result = false;
		break;
	}
}
```
- n = a * b일 때, a b 둘 중 하나는 n의 제곱근보다 작다.
### 3. 에리토스테네스의 체

for(int i=2; i<=Math.sqrt(num); i++) {
