# 기본수학2

## 소수찾기
### 1. 자기자신보다 작은 수로 나누기
```java
for(int i=2; i<num; i++) {
	if(num % i ==0) {
		result = false;
		break;
	}
}
```
### 2. 제곱근
```java
for(int i=2; i<=Math.sqrt(num); i++) {
	if(num % i ==0) {
		result = false;
		break;
	}
}
```

for(int j = i * i; j < Max + 1; j = j + i) {
